# my_customTools

> 统一媒体转码工具集 - 视频转码 & 图片转码 & 文档转换 & 消息推送

## ✨ 功能特性

- **视频转码**：支持 H.264、H.265、VP9 等编码格式
- **图片转码**：支持 JPEG、PNG、WebP、AVIF 等格式转换
- **文档转换**：Word 转 PDF、Excel 转 CSV、PDF 合并拆分
- **动图处理**：GIF、WebP、APNG 格式转换
- **消息推送**：实时消息通知中心，支持普通消息和待办消息
  - 右上角铃铛图标显示未读红点
  - 弹出层展示最近 5 条消息预览
  - 支持标记已读、清空消息
  - 完整消息列表页面，支持筛选和分页
