## 技术约定

### API 响应格式

```typescript
interface ApiResponse<T> {
  code: number;    // 0 表示成功，非 0 表示错误
  msg: string;     // 提示信息
  data: T;         // 数据
}
```

### 错误处理

```typescript
// 统一错误码
const ERROR_CODES = {
  SUCCESS: 0,
  UNAUTHORIZED: 401,
  FORBIDDEN: 403,
  NOT_FOUND: 404,
  VALIDATION_ERROR: 400,
  SERVER_ERROR: 500
};
```

### 任务状态

```typescript
type TaskStatus = 'waiting' | 'uploading' | 'processing' | 'completed' | 'failed' | 'cancelled';
```
