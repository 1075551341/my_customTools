## 遇到的问题与解决

### 问题 1: CORS 跨域限制
- **现象**: 前端请求被后端 CORS 策略阻止
- **原因**: 后端只允许 5173/5174 端口
- **解决**: 更新 `backend/src/config/defaults.ts` 添加 6001 端口

### 问题 2: Vite 代理路径重写
- **现象**: `/api/*` 请求返回 404
- **原因**: rewrite 配置移除了 `/api` 前缀
- **解决**: 移除 vite.config.ts 中的 rewrite 配置

### 问题 3: TypeScript 类型错误
- **现象**: `requireRole` 函数类型不匹配 'super' 角色
- **原因**: 类型定义缺少 'super' 角色
- **解决**: 更新 `backend/src/middlewares/auth.ts` 类型定义

### 问题 4: 滑块验证
- **现象**: 自动化测试无法通过登录
- **原因**: 前端安全机制需要用户交互
- **解决**: 测试时通过 API 直接获取 token 注入
